Project Folder Structure

Frontend (React)

my-app/
│
├── public/
│   └── index.html
│
├── src/
│   ├── components/
│   │     ├── Navbar.js
│   │     ├── Footer.js
│   │
│   ├── pages/
│   │     ├── Home.js
│   │     ├── About.js
│   │

│   │     └── api.js
│   │
│   ├── App.js

│
├── package.json

Backend (Node.js)

server/
│
├── controllers/
│   └── userController.js
│
├── routes/
│   └── userRoutes.js
│
├── models/
│   └── userModel.js
│
├── config/
│   └── db.js
│
├── server.js
└── package.json

Explanation

  • components: Reusable UI parts
  • pages: Full screens
  • services: API calls
  • controllers: Logic handling
  • routes: API endpoints
  • models: Database structure
