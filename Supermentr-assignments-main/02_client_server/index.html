Client-Server Architecture

What is Client?

A client is a device or application that requests services or resources from a server. Example: Web browser (Chrome, Edge)

What is Server?

A server is a system that provides resources, data, or services to clients over a network.

Architecture Diagram

        [ Client (Browser) ]
                |
                | Request (HTTP)
                ↓
        [ Internet / Network ]
                |
                ↓
        [ Server ]
                |
                | Response (HTML, CSS, JS)
                ↓
        [ Client Displays Web Page ]

What Happens When You Open a Website?

  1. User enters URL in browser
  2. DNS converts domain name to IP address
  3. Browser sends HTTP request to server
  4. Server processes request
  5. Server sends back response (HTML, CSS, JS)
  6. Browser renders the webpage
